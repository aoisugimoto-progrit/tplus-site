# グループD: PER解説のリライト

## 📋 概要
- **実装時間**: 1時間
- **リスク**: 中
- **効果**: 客観性の向上、情報の整理

---

## ⚠️ 徹底禁止事項

- **人名の削除**: 片山晃氏、田端氏、清原氏などの個人名はすべて削除
- **引用表現の削除**: 「〜さんの言葉」「〜氏の解説」といった表現を削除
- **口語・感情的表現の削除**: 「激怒案件です」などの感情的な表現を削除
- **不要なエピソードの削除**: 「動画で紹介していた」等の背景説明を削除

---

## ✅ リライト方針

### D-1: PERセクション全体の構造

**対象セクション**: 企業分析詳細編②スクリーニング編 - PER

**削除する要素**:
- 個人名への言及
- 「〜氏が教える」などの引用形式
- 口語的な説明（「激怒案件」など）

**維持する要素**:
- PERの定義
- トヨタと日産の比較例
- 札幌ビールの例（わかりやすいため）
- PERが高い/低い業界の説明

---

### D-2: 具体的なリライト例

**変更前**（人名を含む例）:
```
片山晃氏は「PERだけで判断するのは危険」と指摘しています。
```

**変更後**（客観的な表現）:
```
PERだけで判断するのは危険です。業界特性や成長性を総合的に見る必要があります。
```

---

**変更前**（口語的な表現）:
```
これ、激怒案件ですよ。PERが低いからって飛びつくと痛い目に遭います。
```

**変更後**（客観的な表現）:
```
PERが低いという理由だけで投資判断を下すのは危険です。業績悪化や構造的な問題を抱えている可能性があります。
```

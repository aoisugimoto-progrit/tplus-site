# グループE: PBR解説のリライト

## 📋 概要
- **実装時間**: 1時間
- **リスク**: 中
- **効果**: 客観性の向上、情報の整理

---

## ⚠️ 徹底禁止事項

- **人名の削除**: すべての個人名を削除
- **引用表現の削除**: 「〜さんの言葉」などを削除
- **口語・感情的表現の削除**: カジュアルすぎる表現を削除

---

## ✅ リライト方針

### E-1: PBRセクション全体の構造

**対象セクション**: 企業分析詳細編②スクリーニング編 - PBR

**削除する要素**:
- 個人名への言及
- 「〜氏の解説によると」などの引用形式
- 不要なエピソード

**維持する要素**:
- PBRの定義（「財布の値段」理論）
- PBR = 株価 ÷ 1株あたり純資産（BPS）
- PBR 1倍割れの意味
- PBRが機能しない場合の説明

---

### E-2: 具体的なリライト例

**変更前**（人名を含む例）:
```
田端氏が教える「財布の値段」理論は、PBRを理解する上で非常にわかりやすい例えです。
```

**変更後**（客観的な表現）:
```
PBRは「財布の値段」理論で理解できます。1万円入った財布を5千円で買えるなら割安、2万円で買うなら割高という考え方です。
```
